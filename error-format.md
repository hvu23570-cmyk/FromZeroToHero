# API Error Format
{
    "error": {
        "code": "MÃ_LỖI",
        "message": "Thông báo cho người dùng"
    }
}
### Các mã lỗi quy định:
- VALIDATION_ERROR: Dữ liệu gửi lên thiếu hoặc rỗng (400)
- NOT_FOUND: Không tìm thấy Task hoặc tài nguyên (404)
- UNAUTHORIZED: Token không hợp lệ hoặc hết hạn (401)
